# Sales Forecasting Using Historical Retail Data

## Project Description
This project focuses on forecasting future retail sales using historical sales data from the Corporación Favorita dataset. The goal is to analyze sales trends and seasonality, build forecasting models, and demonstrate their application in inventory planning.

## Dataset
- Corporación Favorita Store Sales Dataset (Kaggle)

## Technologies
- Python
- pandas, numpy
- matplotlib
- scikit-learn
- statsmodels
