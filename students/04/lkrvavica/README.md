## Opis projekta

### Koji URL scrapate i zašto
Skripta scrapa https://meteo.hr - službenu stranicu Državnog hidrometeorološkog zavoda Hrvatske. 
Cilj je dohvatiti vremensku prognozu i pomoću AI-ja (Google Gemini) pretvoriti ju u **jednostavan, 
razumljiv i prijateljski tekst** - kao da ga čita radijski voditelj. Umjesto tehničkih termina 
i sirovih podataka, dobijete prognozu u prirodnom jeziku.

### Kako pokrenuti skriptu
pokrenuti marimo: marimo edit /workspaces/2025-sci-prog/students/04/exercise/lkrvavica/scraper.py
dodati api ključeve za steel api i google ai api u env u ovaj folder


### Api ključevi:
```
   STEEL_API_KEY=tvoj_steel_api_key_ovdje
   GOOGLE_API_KEY=tvoj_google_api_key_ovdje
```
